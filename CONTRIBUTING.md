# Contributing to RealNeed

Thank you for helping improve an evidence-first, open-source tool.

- Do not add mock evidence, fabricated URLs, or fallback opportunity generation.
- Never commit `.env*`, user data, report links, source bodies, logs, or API keys.
- Run `npx tsc --noEmit`, `npm run lint`, `npm test`, `npm run build`, and `npm run db:check` before a pull request.
- Changes touching evidence policy, providers, credentials, jobs, or report links need focused tests.
- Report security issues privately according to `SECURITY.md`.
